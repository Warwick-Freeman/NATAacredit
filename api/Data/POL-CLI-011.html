Patient Data Storage Policy

Authorised by: Site Director
Standard: AASM S-14

1. Purpose

To ensure patient data and raw sleep study data are stored for the required minimum period and protected appropriately.

2. Policy

All patient data collected at this service location, including raw data (excluding video) from all sleep tests (PSG, MSLT, HSAT), must be stored for a minimum of five (5) years, or as required by applicable law if longer.

3. Procedure

Retention Requirements

  1. Raw sleep study data (excluding video) stored for a minimum of five (5) years, or longer as required by law.
  2. Data must include all information necessary for study review and re-interpretation.
  3. A secure, reliable, accessible data storage system must be maintained.

Data Security

  1. Patient data stored securely with access controls and encryption where applicable.
  2. Backup procedures are in place to prevent data loss.
  3. Access restricted to authorised personnel.

Release of Records

  1. Electronic copies may be provided to other treating physicians per a patient's written request for release, in compliance with HIPAA.
  2. All record releases are documented.

Data Destruction

  1. Data must not be destroyed before the end of the required retention period.
  2. When destroyed, PHI must be rendered unrecoverable.
  3. Data destruction is documented.
