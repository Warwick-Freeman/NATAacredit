1. Purpose ASA 5.5.4

To define how home sleep studies are selected, set up, analysed and integrated into a package of care.

2. Patient Selection

  1. Sleep physician triages each referral; home study used to rule in (Type 2, 3, 4) and rule out (Type 2) suspected OSA per ASA Guidelines.
  2. Exclusion: significant cardiopulmonary or neuromuscular comorbidity, suspected non-respiratory sleep disorder, prior negative HSAT with persisting symptoms - escalate to attended study.
  3. Limitations of Type 3/4 devices explained to patient.

3. Setup

  • Conducted in a private set-up room or by trained technologist in patient's home.
  • Device hygiene per infection control SOP; lot/serial recorded.
  • Patient instruction: device fitting, sensor placement, troubleshooting, return arrangements.

4. Analysis

Type 2 study: at least 1.5 h analysis time per study. Full raw-signal disclosure retained. Re-study or escalate to attended PSG if technical failure or clinical/study discordance.

5. Onward Referral

Where the service does not also offer attended studies, a documented arrangement exists with an attended-PSG service (SOP-QMS-011).

6. Records

FRM-001 referral; device tracking; analysis output; report template FRM-025.

7. References

ASA Standard cl. 5.5.4.1 - 5.5.4.3, 4.4.3, 4.5; ASA Guidelines.
