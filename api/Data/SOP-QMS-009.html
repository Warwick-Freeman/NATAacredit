1. Purpose ASA 4.10, 4.11

To eliminate the cause(s) of identified nonconformities (corrective action) and the cause(s) of potential nonconformities (preventive action).

2. Definitions

  • Immediate action - taken at the time of NC to mitigate the immediate effect.
  • Corrective action - addresses the root cause of an actual NC.
  • Preventive action - addresses the root cause of a potential NC.

3. Procedure

  1. NC or risk identified; FRM-008 raised.
  2. Root-cause analysis using 5-Whys or fishbone template; multiple contributory factors documented.
  3. Action plan defined with owner, target date and success criteria.
  4. Action implemented; resources tracked.
  5. Effectiveness review by Quality Manager at the planned re-check date - usually 1 - 3 months after implementation.
  6. If effective, CAPA closed; if not, plan revised and re-implemented.

4. Preventive Action Triggers

Review of QMS data including indicator trends, complaint patterns, supplier-performance shifts, equipment-verification drift and external publications. Quality Manager initiates preventive CAPA without an active NC.

5. Records

CAPA register; FRM-008; effectiveness verification.

6. References

ASA Standard cl. 4.10.1 - 4.11.2.
